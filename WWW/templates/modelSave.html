Help: Saving a Working "Model"

What is a Model?

A model is the output of a PLS or PCA analysis on data. By saving this data for later use, you can run analysis against samples taken previously. Saveing a model here will save the output of the last PLS or PCA that has been done during this sensing session.

Save isnt avaliable?

Before you can save your model, you must first peform PLS or PCA on a set of scans. The result of that analysis is saved.
